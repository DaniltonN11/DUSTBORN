# Dustborn — Wiki do Projeto

Bem-vindo à wiki do projeto Dustborn. Esta wiki reúne o GDD, arquitetura, protótipo, assets, UI, roadmap e instruções de contribuição.

Seções:
- `Overview.md` — Visão geral do projeto e objetivos.
- `GDD.md` — Documento de design (GDD v1.1) resumido e links para o arquivo completo `GDD_Dustborn_v1.1.md`.
- `Architecture.md` — Arquitetura do Colossus e padrões de rede (multi‑engine).
- `Prototype_Godot.md` — Instruções e descrição do esqueleto Godot criado em `godot_project/`.
- `Assets.md` — Lista de assets por sprint (link para `assets_by_sprint.csv`) e fluxo de pipeline de arte.
- `UI.md` — Wireframes, hi‑fi e instruções para usar `wireframes/mockup_interactive.html`.
- `Roadmap.md` — Sprints, milestones e prioridades para MVP.
- `Executive_Summary.md` — Versão da wiki do sumário executivo (referência a `EXECUTIVE_SUMMARY.md`).
- `FAQ.md` — Pontos de decisão e checklist para stakeholders (espelho de `FAQ_DEVELOPMENT.md`).
- `HowToRun.md` — Passos rápidos para abrir o projeto no Godot 4.1+ e testes básicos.
- `Contributing.md` — Guia de contribuição, convenções e padrão de commits.

Para editar esta wiki localmente, abra o diretório `wiki/` no editor e modifique os arquivos Markdown.
